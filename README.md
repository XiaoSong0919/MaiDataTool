# MaiDataTool

用于对从HDD批量提取的Simai3谱进行筛选并重命名的工具

推荐个谱面合集：https://space.bilibili.com/21753216/article

-----------------------------------------------------------------

需要在“文件夹/分类文件夹/谱面文件夹”的文件结构下运行

举个例子，现需要对“maimai”分类的谱面进行筛选，那么整体的文件结构为

D:/Output/maimai/xxxx

其中xxxx为存放maidata.txt的文件夹，

输入文件夹为"D:/Output"

-----------------------------------------------------------------
目前已实现功能：

## 筛选

可以根据以下条件对谱面进行筛选

·歌曲名称

·歌曲ID

·发布版本

·BPM

·谱师

·谱面版本（SD/DX）
